# Synopia

A landing page for **Synopia** — a platform to coordinate the lending of artworks
between museums for temporary exhibitions: loan requests, condition reports,
insurance, availability, and transit, all in one shared record.

## Stack

A single self-contained `index.html` — no build step, no dependencies.
Fonts (Fraunces, Instrument Sans) load from Google Fonts.

## Run

Open `index.html` in any browser, or serve the directory:

```sh
python3 -m http.server
```
